# Contrast

This is a pet project. Sometimes I even work on it!
![Screenshot Dark](/screenshot-dark.png)
![Screenshot Light](/screenshot-light.png)

## Features

- Multiple repositories
- Staged and Unstaged files
- Syntax highlighting
- Magic scrolling
- Sub-line diffing
- Support for Atom themes

## To Use

- npm install
- npm start
